== Description ==
 
This plugin will help editors create special folders on specific topics or events. You need just to insert the ID of each posts and they will appear automatically in the folder.
 
Major new features in Special Folder include:
 
* Help editors create special folders on specific topics or events.
* Just insert the IDs of posts and they will appear automatically in the folder.
* Custom css.
* You can edit the template.
 
Developed by EDC [(e-Da`wah Committee)](https://www.edc.org.kw). For new plugins, please visit this page: [Islam.com.kw](https://www.islam.com.kw).
 
== Installation ==
 
* Upload Special Folder plugin to your blog, Activate it.
* Go to Special Folder from menu.
* Copy blue code like this: posts[*] and past code in your post or page.
